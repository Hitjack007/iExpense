# iExpense

iExpense is an iOS app built to hel you track expenses. This app features permanent meory - so your expenses are always saved. Developed during Paul Hudsonâ€™s [100 Days of Swift](https://www.hackingwithswift.com/100) course, it showcases UserDefaults integration with SwiftUI.

## Features
- Track Expenses
- Name Expenses, Specify Business or Personal and the Amount Spent
- UserDefaults Memory
- Clean, responsive UI supporting light/dark modes

## Getting Started

1. Download or clone this repo.
2. Open the project folder in Xcode.
3. Press **Cmd + R** to build and run on the simulator or device.
